# la-planque-sonore

## Installation

### Pour les dévelopeurs de la muerte

```bash
$ npm install
$ bower install
$ grunt
```

### Pour les mecs qui veulent juste taper sur leur clavier

Ouvrir le fichier `dist/index.html` dans un navigateur web
